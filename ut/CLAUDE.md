# CLAUDE.md

## What this is

An interactive scrollytelling web page for a job application to Uncharted Territories (Tomas Pueyo's newsletter). Hosted at `thiska.works/ut`. The page itself is the proof of work.

## Deployment

- Hosted on Hostinger, auto-deploys on push to `master`
- All files live in the `/ut` directory

## Tech stack

- Plain HTML / CSS / JS (no framework unless a specific need arises)
- Scrollama.js for scroll-driven interactions
- D3.js or Chart.js for data visualization
- Mobile-responsive (reviewer may be on phone)

## Constraints

- Single page, self-contained
- Must feel polished — quality over quantity
- Keep it lightweight and fast-loading

Mathis Kakal — Paris-based developer, former mediatech startup CTO, full-stack (assembly to Flutter), UX competition background, long-time UT reader. Building this to show what UT articles could become as interactive experiences.
